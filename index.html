    Driveway Security System

    Description

    This project was built using Tinkercad’s Arduino simulator to develop an IoT gateway security system. The goal was to simulate a smart access control mechanism that responds to password input using a combination of visual, audio, and mechanical components.

    Key Features

    Password-Protected Access: User must enter a correct password to trigger the system.

    Visual & Audio Feedback: Green/red LED indicators and piezo buzzer signals for access status.

    Servo Motor Gate Simulation: Simulates a gate opening upon successful access.

    LCD Feedback Display: Dynamic messages show "Access granted" or "Access denied" based on input.

    Tools and Technologies

    Arduino, C++, Tinkercad

    Weather API (Live Weather Report)

    Description

    This project demonstrates how to connect to a third-party weather API using JavaScript to fetch and display real-time weather data based on user input. The user enters a city name and clicks the “Get Weather” button. The program retrieves the current weather conditions and visually simulates the temperature.

    Key Features

    Live Weather Data: Fetches and displays city-specific weather information including temperature, description, and icon.

    Visual Temperature Feedback: Slider displays temperature, while background color dynamically shifts (red for hot, orange/yellow for warm, light/dark blue for cold).

    Weather Icons: Visual representation of current weather conditions (e.g. sun, clouds, rain).

    Tools and Technologies

    HTML5, CSS3, JavaScript, JSON, Fetch API
